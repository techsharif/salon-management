# Confluence Documents Local Storage

This directory contains local copies of all Confluence documents for context and synchronization.

## Structure

- `docs/` - General documentation (API docs, design docs, etc.)
- `requirements/` - Requirements documents

## File Naming Convention

Format: `[DOC-TITLE-SLUG].md`

Example:
- `api-documentation.md`
- `system-architecture.md`
- `user-authentication-requirements.md`

## Purpose

- Maintain local context for AI assistance
- Enable two-way sync with Confluence
- Track document versions and changes
- Reference documents without API calls
